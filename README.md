- Vite
- React
- shadcn/ui

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/web-loop-content.git
cd web-loop-content
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
